# Gimp Selection Feature Plugin QGIS

Work with GIMP. Add image, (view of canvas) or source, from QGIS to GIMP. Add selection area from GIMP to a polygon layer in QGIS.

*Use DBUS libray. Not implement in WINDOWS.*
